# Quiz Diritto Penale - GitHub Pages

## File da caricare su GitHub
Carica questi file nello stesso repository:
- `index.html`
- `style.css`
- `script.js`
- `questions.json`

## Come pubblicarlo
### Metodo più semplice
1. Crea un nuovo repository su GitHub
2. Carica tutti e 4 i file nella root del repository
3. Vai su **Settings**
4. Vai su **Pages**
5. In **Build and deployment**, scegli:
   - **Source** → `Deploy from a branch`
   - **Branch** → `main`
   - **Folder** → `/(root)`
6. Salva
7. Dopo poco GitHub ti darà il link del sito

## In alternativa
Puoi mettere i file dentro una cartella `docs/` e poi in Pages scegliere `/docs`.

## Nota
Il quiz:
- mostra 1 domanda alla volta
- permette di andare avanti e indietro
- corregge tutto alla fine
- può mescolare l'ordine delle domande
- mantiene le risposte originali con corretta sempre sulla A nel database
